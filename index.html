Guia de Configuração

Siga este guia passo a passo para configurar o aplicativo

Requisitos do Sistema

Antes de começar, certifique-se de ter:

  • Python 3.7 ou superior
  • Conta Google com acesso ao Google My Business
  • Acesso ao Google Cloud Console
  • Chave de API do Google Gemini

1. Criar Projeto

Acesse o Google Cloud Console e crie um novo projeto

2. Ativar APIs

Ative as seguintes APIs:

  • My Business Business Information API
  • My Business Account Management API

3. Configurar OAuth

Configure a tela de consentimento OAuth e crie credenciais OAuth 2.0

4. Baixar Credenciais

Baixe o arquivo JSON e renomeie para client_secrets.json

Configuração da API Gemini

1. Acesse o Google AI Studio

2. Gere uma chave de API

3. Configure como variável de ambiente:

GEMINI_API_KEY=sua_chave_api_aqui

1. Criar Ambiente Virtual

python -m venv venv .\venv\Scripts\Activate.ps1 # Windows source venv/bin/activate # Linux/macOS

2. Instalar Dependências

pip install flask google-auth google-auth-oauthlib google-auth-httplib2 google-api-python-client requests python-dotenv google-generativeai

3. Executar Aplicativo

python src/main.py

Perguntas Frequentes

Soluções para os problemas mais comuns

Este erro indica que o Flask não está instalado ou o ambiente virtual não está ativado. Certifique-se de ativar o ambiente virtual e instalar as dependências:

.\venv\Scripts\Activate.ps1 pip install flask

Você precisa baixar o arquivo de credenciais OAuth 2.0 do Google Cloud Console e colocá-lo na pasta raiz do projeto com o nome 'client_secrets.json'.

Adicione uma chave secreta ao seu aplicativo Flask. No arquivo main.py, adicione:

app.secret_key = "sua_chave_secreta_aqui"

Durante o desenvolvimento, adicione seu e-mail como testador na tela de consentimento OAuth ou use a opção "Avançado" para prosseguir.

Certifique-se de que os arquivos HTML estão na pasta 'templates' e configure o Flask para usar o caminho correto:

template_dir = os.path.abspath('templates') app = Flask(__name__, template_folder=template_dir)

Sobre o Projeto

O Gerenciador de Avaliações Google com IA é um aplicativo web desenvolvido para ajudar empresários e profissionais a gerenciar suas avaliações no Google My Business de forma mais eficiente.

Utilizando a integração com a API do Google My Business e a inteligência artificial do Google Gemini, o aplicativo oferece sugestões automáticas de respostas personalizadas, economizando tempo e mantendo um alto padrão de atendimento ao cliente.
